# 如何做好布局

> 前提: 每隔CSS属性,  每个 html标签 深入了解

- 做好布局需要考虑的方面
  1. 合理的布局结构(符合规范,合适标签,少量的嵌套,结构合理,复用CSS)
  2. 严格按照设计稿开发
  3. 符合项目要求兼容性
  4. 布局优化,性能优化, SEO优化
  5. 交互效果, 功能开发  (表单验证)
