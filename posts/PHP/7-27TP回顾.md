## 登录模块

- 登录  退出登录  登录权限 密码加密

- 登录流程

  -  前台  表单  ---ajax---> Login控制器   -----> Login模型   checkLogin验证

    ​		----->成功后   :: 跳转到主页

    ​		------>失败后  ::   Login控制器 ---> 前台 用户提示

  - 退出登录:

    1. 清理登录状态  Session::clear()   Session::destory()
    2. 跳转至登录页

  - 访问权限

    1. 登录成功    Session::set('isLogin',true)

    2. 需要登录页面, 进行权限判断

       if(!Session::get('isLogin')){

       ​	return   $this->error('没有登录权限','URL')

       }
